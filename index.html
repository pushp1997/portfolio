Amazon Product Reviews Scraping

The tool takes a link (any Amazon product page) as an input, and scrapes all the customer reviews to that product off the Amazon website and collects the data in a Pandas Dataframe. The data is then saved in the project folder in a CSV file format. This data can later be used for Sentiment Analysis of Customers related to that product.

Language and Frameworks Used:
Python, Requests Library, BeautifulSoup Library, Pandas Library, Jupyter Notebook
